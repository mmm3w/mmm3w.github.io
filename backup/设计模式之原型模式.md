### 原型模式

##### 个人理解

创建给定类的实例过程过于昂贵和复杂时，使用原型模式。
允许通过复制现有类的实例来创建新的实例。有时候，复制对象比创建新对象更有效。

##### 实例代码

简单的实现只是浅克隆

```java
public class Product implements Cloneable{
　　public Object clone(){
　　　　Object object = null;
　　　　try {
　　　　　　object = super.clone();
　　　　} catch (CloneNotSupportedException exception) {
　　　　　　System.err.println("Not support cloneable");
　　　　}
　　　　return object;
    }
    ……
}
```
